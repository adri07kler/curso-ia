3.6 Crear una cadena de prompts

Módulo 3 · Ejercicio

Objetivo: Practicar la creación de cadenas de prompts para un problema real.

Enunciado

Dado el requerimiento:

"Dashboard para usuarios que muestra sus últimas 10 transacciones, su saldo actual, y un gráfico de gastos por categoría del mes actual. Solo datos de los últimos 30 días. Debe actualizarse cada 5 minutos."

Crea una cadena de 6 prompts:

  1. Prompt 1: Analiza el requerimiento y detecta ambigüedades
  2. Prompt 2: Convierte en especificación clara
  3. Prompt 3: Propón plan de implementación por pasos
  4. Prompt 4: Implementa solo el primer paso
  5. Prompt 5: Genera pruebas para validar
  6. Prompt 6: Revisa críticamente el resultado

Incluye también el context packet necesario.
