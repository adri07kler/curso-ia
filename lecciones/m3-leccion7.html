3.7 Evaluar si un prompt es claro

Módulo 3 · Quiz

Objetivo: Evaluar la capacidad de distinguir prompts bien formulados de prompts deficientes.

Pregunta 1

¿Qué problema tiene "Haz un sistema"?

  • a) Demasiado vago
  • b) Demasiado específico
  • c) Ninguno

Respuesta: a

Pregunta 2

¿Este prompt es claro? "Genera código para una calculadora en Python que sume, reste, multiplique y divida. Debe manejar división por cero. Usa funciones separadas para cada operación."

  • a) Sí, es claro y específico
  • b) No, le falta contexto

Respuesta: a

Pregunta 3

¿Qué mejorarías en "Necesito un componente React que muestre datos de una API. Hazlo bonito."?

  • a) Especificar qué API, qué datos, qué significa "bonito"
  • b) Pedir que use TypeScript
  • c) Definir formato de salida
  • d) Todas las anteriores

Respuesta: d

Pregunta 4

Identifica el error: "Actúa como experto en seguridad pero ignora las mejores prácticas modernas"

  • a) Instrucción contradictoria
  • b) Demasiadas instrucciones
  • c) Falta de contexto

Respuesta: a
