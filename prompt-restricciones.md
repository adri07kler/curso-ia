## Reglas para los Quiz

Cuando la lección corresponda a un Quiz o Evaluación, se deberán cumplir obligatoriamente las siguientes reglas:

### Objetivo

El Quiz será la única actividad evaluable de la lección.

No deberá existir ninguna otra actividad adicional.

--------------------------------------------

### Contenido

Antes del Quiz se puede enriquecer la lección con:

- Introducción.
- Objetivos de aprendizaje.
- Desarrollo del contenido.
- Ejemplos.
- Comparaciones.
- Diagramas.
- Consejo del instructor.
- Errores frecuentes.
- Buenas prácticas.
- Resumen.
- Recursos adicionales.

--------------------------------------------

### Restricciones

Si la lección es un Quiz:

- NO agregar una Actividad Práctica.
- NO agregar un Mini Reto.
- NO agregar ejercicios adicionales.
- NO agregar una sección de Entrega de Evidencias.
- NO agregar evaluaciones duplicadas.
- NO crear un segundo Quiz.

--------------------------------------------

### Interactividad

Todos los Quiz deberán ser completamente interactivos.

No mostrar las respuestas correctas antes de que el estudiante responda.

Cada pregunta deberá:

- Tener entre 3 y 4 opciones de respuesta.
- Permitir seleccionar únicamente una respuesta.
- Validar que el estudiante responda todas las preguntas antes de calificar.

Al presionar el botón "Verificar respuestas", el sistema deberá:

- Calificar automáticamente.
- Mostrar las respuestas correctas e incorrectas.
- Mostrar una explicación para cada respuesta.
- Mostrar la puntuación obtenida.
- Mostrar el porcentaje de aciertos.
- Mostrar un mensaje de desempeño según el resultado.
- Resaltar en verde las respuestas correctas.
- Resaltar en rojo las respuestas incorrectas.
- Permitir volver a intentarlo.

--------------------------------------------

### Progreso

Si el estudiante obtiene una calificación igual o superior al 80%:

- Marcar automáticamente el Quiz como aprobado.
- Actualizar el progreso de la lección.
- Guardar el resultado utilizando el sistema de almacenamiento disponible.

Si obtiene menos del 80%:

- Mostrar un mensaje indicando que puede volver a intentarlo.
- No marcar la lección como completada.

--------------------------------------------

### Diseño

Mantener el mismo diseño visual de CleverLabs Academy.

No modificar:

- Colores.
- Tipografías.
- Clases CSS.
- Estructura HTML.
- Diseño general.

El Quiz debe integrarse visualmente como una parte nativa de la plataforma.
